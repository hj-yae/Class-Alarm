
< 수업 알리미 프로그램 사용 설명서 >

1. 프로그램 소개
  '수업 알리미'는 학교나 학원에서 수업 시간표를 관리하고, 수업 시작 전에 알림을 제공하는 프로그램입니다. 
  이 프로그램을 통해 시간표 관리, 공지사항 작성, 수업 시작 알림 등의 기능을 사용할 수 있습니다.

2. 주요 기능
  - 시간표 관리
  - 공지사항 관리
  - 수업 시작 알림
  - 실시간 시계 표시

3. 프로그램 시작
  프로그램을 실행하면 메인 화면이 나타납니다. 메인 화면에는 현재 날짜, 시간, 다음 수업 정보, 남은 시간 등이 표시됩니다.

4. 시간표 관리
  (1) 메인 화면에서 '수업 설정 열기' 버튼을 클릭합니다.
  (2) '시간표 편집' 버튼을 클릭하여 시간표 편집 창을 엽니다.
  (3) 각 요일과 교시에 해당하는 과목을 선택합니다.
  (4) 변경 사항을 저장하려면 '저장' 버튼을 클릭합니다.

5. 공지사항 관리
  (1) 메인 화면에서 '수업 설정 열기' 버튼을 클릭합니다.
  (2) '공지사항 편집' 버튼을 클릭하여 공지사항 편집 창을 엽니다.
  (3) 각 과목에 대한 공지사항을 입력합니다.
  (4) 교사의 메시지를 입력할 수 있습니다.
  (5) '모두 저장' 버튼을 클릭하여 변경 사항을 저장합니다.

6. 수업 시간 설정
  (1) 메인 화면에서 '수업 설정 열기' 버튼을 클릭합니다.
  (2) '수업 시간 편집' 버튼을 클릭하여 수업 시간 편집 창을 엽니다.
  (3) 각 교시의 시작 시간과 종료 시간을 설정합니다.
  (4) '저장' 버튼을 클릭하여 변경 사항을 저장합니다.

7. 프로그램 설정
  메인 화면에서 '프로그램 설정' 버튼을 클릭합니다. 
  다음 설정을 변경할 수 있습니다:
    - 수업 시간표 파일 경로
    - 알람 소리 파일 경로
    - 알람 시간 (수업 시작 전 몇 분 전에 알림을 받을지)
    - 알람 소리 크기

8. 알림 기능
  설정된 알람 시간에 따라 다음 수업 시작 전에 알림음이 재생됩니다.
  알림음이 재생될 때 화면의 '남은 시간' 표시가 빨간색으로 변경됩니다.

9. 프로그램 종료
  메인 창의 닫기 버튼을 클릭하면 프로그램 종료 확인 메시지가 표시됩니다. '예'를 선택하면 프로그램과 모든 관련 창이 종료됩니다.

10. 수업 시간표 파일 작성 방법
  수업 시간표 파일은 Excel(.xlsx) 형식으로 작성해야 합니다. 이 파일은 프로그램이 시간표 정보를 읽어오는 데 사용됩니다. 다음 지침에 따라 파일을 작성해 주세요:

  10.1 파일 구조
    (1) 파일명: school_schedule.xlsx (기본 설정)
    (2) 필요한 시트:
      - '수업명': 과목 목록을 포함
      - '시간표': 요일별, 교시별 수업 시간표를 포함

  10.2 '수업명' 시트 작성
    A열의 제목을 '수업명'으로 설정합니다. A열에 모든 과목명을 나열합니다.

  10.3 '시간표' 시트 작성
    A1 셀에 '교시'를 입력합니다. B1:F1 셀에 각각 '월요일', '화요일', '수요일', '목요일', '금요일'을 입력합니다. A열에 교시 번호를 입력합니다 (1교시부터 시작).
    B열부터 F열까지 각 요일의 수업을 입력합니다.

  10.4 주의사항
    - '수업명' 시트의 과목명과 '시간표' 시트에 입력한 과목명이 정확히 일치해야 합니다.
    - 수업이 없는 시간에는 '없음'을 입력하거나 빈 칸으로 두세요.
    - 최대 8교시까지 지원됩니다. 필요한 교시만큼만 입력하세요.
    - 엑셀 파일을 저장할 때 반드시 .xlsx 형식으로 저장하세요.

  10.5 파일 적용 방법
    작성한 엑셀 파일을 프로그램이 있는 폴더에 저장합니다. 프로그램의 '프로그램 설정'에서 '수업 시간표 파일 경로'를 확인하고, 필요하다면 수정합니다. 프로그램을 재시작하거나, '수업 설정 열기'에서 시간표를 다시 불러옵니다.

11. 주의사항
  - 프로그램을 처음 실행할 때는 기본 설정으로 시작됩니다. 정확한 사용을 위해 시간표와 수업 시간을 반드시 설정해주세요.
  - 알람 소리 파일이 올바르게 설정되어 있는지 확인해주세요.
  - 공지사항은 해당 날짜의 수업에만 표시됩니다. 필요한 경우 매일 업데이트해주세요.

12. 문제 해결
  - 프로그램이 제대로 작동하지 않는 경우, 모든 설정이 올바르게 되어 있는지 확인해주세요.
  - 알림음이 들리지 않는 경우, 알람 소리 파일 경로와 볼륨 설정을 확인해주세요.
  - 기타 문제가 발생하면 프로그램을 재시작하거나, 개발자에게 문의해주세요.
